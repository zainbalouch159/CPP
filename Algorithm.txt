SORT (ascending order)

```cpp
sort(arr , arr+7);
```

Example

```cpp
int arr[5]={5,1,4,2,3};
sort(arr, arr+5);
```

Output

```
1 2 3 4 5
```

---

COMPLETE EXAMPLE

```cpp
#include <iostream>
#include <algorithm>
#include <numeric>
using namespace std;

int main(){

int arr[7]={1,253,252,2,5,25,4};

sort(arr , arr+7);

cout<<"Max = "<<*max_element(arr,arr+7)<<endl;
cout<<"Min = "<<*min_element(arr,arr+7)<<endl;
cout<<"Count of 5 = "<<count(arr,arr+7,5)<<endl;
cout<<"Sum = "<<accumulate(arr,arr+7,0)<<endl;

return 0;
}
```
