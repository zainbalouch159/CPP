
STL KA STRUCTURE

STL ke 3 main parts hote hain

Container
data store karta hai

Iterator
data tak pohanchne ka tareeqa

Algorithm
data par operation karta hai

---

VECTOR

Header

```cpp
#include <vector>
```

Declare

```cpp
vector<int> v;
```

Insert

```cpp
v.push_back(10);
v.push_back(20);
v.push_back(30);
```

Beech me insert

```cpp
v.insert(v.begin()+1 , 15);
```

Access

```cpp
cout<<v[0];
```

Print

```cpp
for(int x : v)
cout<<x<<" ";
```

LIST

Header

```cpp
#include <list>
```

Declare

```cpp
list<int> l;
```

Insert

```cpp
l.push_back(10);
l.push_back(20);
l.push_back(30);
```

Front me insert

```cpp
l.push_front(5);
```

Beech me insert

```cpp
auto it = l.begin();
advance(it , 1);
l.insert(it , 15);
```

Print

```cpp
for(int x : l)
cout<<x<<" ";
```

---

VECTOR INSIDE MAP

Declare

```cpp
map<string,vector<int>> marks;
```

Insert

```cpp
marks["Zain"] = {90,85,80};
```

Print

```cpp
for(auto x : marks)
{
cout<<x.first<<" | ";
for(int m : x.second)
cout<<m<<" ";
cout<<endl;
}
```

ITERATOR

Iterator container ke elements ko point karta hai
ye pointer jaisa hota hai

Declare

```cpp
vector<int>::iterator it;
```

Start position

```cpp
it = v.begin();
```

End position

```cpp
v.end();
```

Value print

```cpp
cout<<*it;
```

Loop

```cpp
for(it=v.begin(); it!=v.end(); it++)
cout<<*it<<" ";
```

---

IMPORTANT ITERATOR FUNCTIONS

Iterator aage move

```cpp
advance(it , 2);
```

Next position

```cpp
next(it);
```

Previous position

```cpp
prev(it);
```

Distance between iterators

```cpp
distance(v.begin(), v.end());
```

ALGORITHMS

Header

```cpp
#include <algorithm>
```

Sort

```cpp
sort(v.begin(), v.end());
```

Find

```cpp
auto it = find(v.begin(), v.end(), 20);
```

Reverse

```cpp
reverse(v.begin(), v.end());
```

Count

```cpp
count(v.begin(), v.end(), 5);
```

Maximum

```cpp
*max_element(v.begin(), v.end());
```

Minimum

```cpp
*min_element(v.begin(), v.end());
```

---

CONTAINER COMPARISON

Vector

fast random access
middle insert slow

List

fast insert delete
random access nahi

Set

unique values
automatic sorting

Map

key value pair
sorted by key

Array

fixed size
fastest access

ZAROORI BAAT

STL ko samajhne ka golden rule

Container
data store karta hai

Iterator
data tak pohanchta hai

Algorithm
data par operation karta hai

---

SHORT SUMMARY

vector → dynamic array
list → linked list
set → unique sorted values
map → key value pair
array → fixed size container

iterator → pointer jaisa traversal tool
algorithm → ready made functions
